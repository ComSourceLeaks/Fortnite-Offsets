# Fortnite-Offsets

## 📌 Fortnite Offsets
A simple repository to store and update Fortnite offsets.

---

## 📂 Offsets
```yaml
# Offsets.txt (Example Format)
Offsets:
  PlayerBase: 0x12345678
  Health: 0x87654321
  Shield: 0x11223344
  Ammo: 0x55667788
```

---

## 🔄 Updating Offsets
1. Open `Offsets.txt`.
2. Get The Offset Needed
3. Go To Your Source And Replace The Offset To The New One
